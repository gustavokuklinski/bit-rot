# Bit Rot
Bit Rot is a zombie survivor game.

**Basic controls**:
- **W/A/S/D**: Walk
- **SHIFT + Click**: Shoot/Attack
- **Click + Drag**: Place on Belt/Backpack or Drop
- **1/2/3/4/5**: Equip/Use item from Belt

- **E**: Get item from floor
- **R**: Reload weapon

- **I**: Opens inventory and belt
- **H**: Opens Status
